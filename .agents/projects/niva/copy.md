# Nivå — Copy & Tone of Voice

> Primary language: **Swedish (sv-SE)**
> All in-app copy in Swedish. App Store listing Swedish (primary) + English (secondary).

## Tone Matrix

| Context | Tone | Example |
|---------|------|---------|
| Welcome / onboarding | Warm, encouraging | "Välkommen! Låt oss hitta ditt nästa hem." |
| Financial data | Confident, clear | "Din KALP-marginal: 4 200 kr/mån" |
| Good news (green) | Reassuring | "Bra ekonomisk hälsa — föreningen sparar bra." |
| Warning (yellow) | Honest, not alarming | "Nära din budget. Dubbelkolla avgiften." |
| Bad news (red) | Direct, supportive | "Hög skuldsättning — kan vara riskabelt." |
| Errors | Helpful, never blaming | "Något gick fel. Försök igen." |
| Paywall | Value-first, not pushy | "Se hela analysen för 99 kr" |
| Empty states | Guiding | "Klistra in en länk från Hemnet för att börja." |

## Swedish Number Formatting

```
Currency:    3 500 000 kr (space thousands, "kr" suffix)
Percentage:  3,5 % (comma decimal, space before %)
sqm:         65 m² (space before m²)
Monthly:     12 500 kr/mån (slash-mån suffix)
```

## Button Labels
```
Primary CTA:     "Kom igång" / "Analysera" / "Lås upp"
Secondary:       "Läs mer" / "Jämför" / "Hoppa över"
Destructive:     "Ta bort" / "Radera profil"
Confirmation:    "Ja, ta bort" / "Avbryt"
```

## Error Messages
```
Network:    "Kan inte nå servern. Kolla din uppkoppling och försök igen."
Scraping:   "Kunde inte hämta data från mäklaren. Försök med en annan länk."
AI fail:    "AI-analysen misslyckades. Vi försöker igen automatiskt."
Invalid:    "Ange ett belopp mellan 0 och 10 000 000 kr."
```

## Words to Avoid
- ❌ "Grattis!" — too informal for financial context
- ❌ "Spännande" — overused, means nothing
- ❌ "Innovativ" — tech buzzword
- ❌ "Leverera" — corporate Swedish
- ✅ "Bra" / "Tydligt" / "Tryggt" — honest, simple, warm
